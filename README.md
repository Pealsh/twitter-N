# Twitter 風 SNS（練習用）

X（旧 Twitter）に近い画面で、投稿したり返信したりできる Web アプリです。  
学習や試作向けの Django プロジェクトです。

## 中身

- 投稿（文字・画像・動画）
- 返信、返信への返信
- いいね
- フォロー
- 通知
- プロフィール（名前・自己紹介・アイコン）

## 用意するもの

- **Python** 3.13 あたり（このフォルダには `myproject/myenv_new` という仮想環境が入っています）

## 始め方

1. ターミナルでアプリのフォルダへ移動する。

   ```bash
   cd myproject
   ```

2. データベースを用意する（初回だけでよいことが多いです）。

   ```bash
   ./myenv_new/bin/python manage.py migrate
   ```

3. サーバーを起動する。

   ```bash
   ./myenv_new/bin/python manage.py runserver
   ```

4. ブラウザで **http://127.0.0.1:8000/** を開く。

5. アカウントは「新規登録」から作れます。管理画面は **http://127.0.0.1:8000/admin/** です（スーパーユーザーは `createsuperuser` で作ります）。

## フォルダの説明（ざっくり）

| 場所 | 何が入っているか |
|------|------------------|
| `myproject/` | Django の本体（`manage.py` がある場所） |
| `myproject/sns/` | この SNS の機能のコード |
| `myproject/static/` | CSS や画像など |
| `myproject/media/` | ユーザーがアップロードしたファイル（起動後にできます） |

## 注意

- **本番の公開サイト**にはそのまま使わないでください。設定は「開発用」のままです。
- 秘密の鍵（`SECRET_KEY`）などがコードに書いてあるので、公開する前には必ず差し替えが必要です。
